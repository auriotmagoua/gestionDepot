    Le produit "Diso-gestion de l'entreprise smart diso" est une application web conçue pour faciliter la gestion des activités au sein de l'entreprise. Ses principales fonctionnalités sont :
    1. Gestion des présences du personnel :

      2. Utilisation d'un système de pointage par QR code pour enregistrer les arrivées et départs du personnel. Génération de rapports de présence.

    2. Gestion des demandes de permissions :

      3. Permet aux employés de soumettre des demandes de permissions (congés, absences, etc.). Système d'approbation des demandes par les responsables. Historique des demandes de permissions.

    3. Gestion des stagiaires :

      4. Permet d'enregistrer et de suivre les informations des stagiaires. Génération d'attestations de stage pour les stagiaires
